# カンバンTodoアプリ 要件定義

## 概要

タスクをカンバンボード形式で管理できるWebアプリケーション。
ユーザーはタスクを作成し、ステータスごとのカラムにドラッグ&ドロップでタスクを移動させることで、進捗を視覚的に管理できる。

## 対象ユーザー

- 個人でタスク管理を行いたいユーザー

## 機能要件

### タスク管理

- タスクの作成（タイトル、説明）
- タスクの編集（タイトル、説明、ステータス）
- タスクの削除
- タスクの一覧表示（カンバンボード形式）
- タスクの詳細表示

### カンバンボード

- 3つの固定カラムでタスクを分類する
  - **Todo**: 未着手のタスク
  - **In Progress**: 進行中のタスク
  - **Done**: 完了したタスク
- ドラッグ&ドロップによるカラム間のタスク移動
- 各カラム内でのタスクの並び順の管理

### データ永続化

- Cloudflare D1（SQLite）によるデータの永続化

## 非機能要件

### スコープ

- ローカル環境での動作をゴールとする（Cloudflareへのデプロイはスコープ外）
- `wrangler dev` でバックエンドを起動し、ローカルD1（SQLite）を使用する

### 技術スタック

既存のプロジェクト構成に準拠する。

| レイヤー                | 技術                                                                |
| ----------------------- | ------------------------------------------------------------------- |
| フロントエンド          | React (Vite), Ant Design, TanStack Router, TanStack Query, @dnd-kit |
| バックエンド            | Hono (Cloudflare Workers)                                           |
| データベース            | Cloudflare D1 (SQLite)                                              |
| 共有型定義              | shared パッケージ                                                   |
| テスト                  | Vitest, MSW                                                         |
| リンター/フォーマッター | Biome                                                               |

### パフォーマンス

- ドラッグ&ドロップ操作時、UIの即時反映（楽観的更新）

### 画面構成

| パス | 画面           | 説明                                      |
| ---- | -------------- | ----------------------------------------- |
| `/`  | カンバンボード | メイン画面。3カラムのカンバンボードを表示 |

## 用語定義

| 用語           | 説明                                                          |
| -------------- | ------------------------------------------------------------- |
| タスク         | 管理対象となる作業単位                                        |
| カラム         | カンバンボード上のステータス区分（Todo / In Progress / Done） |
| カンバンボード | タスクをカラムごとに表示する画面                              |
